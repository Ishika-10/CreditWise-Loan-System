# CreditWise Loan Approval System

## About the Project
SecureTrust Bank receives hundreds of loan applications daily.
Their manual verification process was slow, biased, and inconsistent —
causing good customers to get rejected and high-risk customers to get approved.

This project builds an intelligent loan approval system using Machine Learning
that automatically analyses applicant details and predicts whether a loan
should be **Approved or Rejected** before final human verification.

## Objective
To learn hidden patterns from historical loan application data and provide
accurate, fast, and unbiased loan approval decisions.

## Dataset
Dataset used from Kaggle — [Download Here](https://www.kaggle.com/datasets/dronabopche/customerdemograph-and-cart-data)
